# Daily UI 001 — The Lab | Sign Up Interface

Este es mi primer challenge de UX/UI desarrollado con HTML, CSS y JavaScript.

Diseñé y construí una interfaz de autenticación enfocada en jerarquía visual, microinteracciones y claridad en la experiencia. La idea fue no quedarme solo en el diseño, sino llevarlo a código y hacerlo funcional.

Incluye:
- Diseño estilo glassmorphism
- Estados hover y microinteracciones
- Inputs con floating labels
- Toggle entre Login y Sign Up
- Simulación de estado de carga
- Vista interactiva para mostrar el código (HTML, CSS y JS)

Un primer paso para integrar diseño e implementación en un mismo proyecto.

# Daily UI 001 — The Lab | Sign Up Interface

This is my first UX/UI challenge built with HTML, CSS and JavaScript.

I designed and developed an authentication interface focused on visual hierarchy, microinteractions and interaction clarity. The goal was to move beyond static design and build a fully functional experience.

Includes:
- Glass-style interface
- Hover states and microinteractions
- Floating label inputs
- Login / Sign Up toggle
- Loading state simulation
- Interactive code view (HTML, CSS and JS)

A first step toward connecting design and front-end implementation.
